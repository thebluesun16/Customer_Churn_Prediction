# Customer Churn Prediction
This project builds an end-to-end customer churn intelligence system that not only predicts churn but also explains why customers are likely to churn and recommends data-driven retention actions using Machine Learning, Explainable AI, and Generative AI.

Objectives
Predict customer churn with high recall
Identify key churn drivers using explainable AI
Segment customers by risk level
Generate AI-powered insights for business teams
Recommend targeted retention strategies
